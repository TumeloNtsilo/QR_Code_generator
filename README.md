Project: QR Code Generator
📝 Description

A sleek web page that generates QR codes instantly from any text or URL. Built with HTML, CSS, and JavaScript, it features a responsive design, smooth transitions, and live QR code generation using the QRServer API
.

⚙️ Features

Instant QR Code Generation: Dynamically creates QR images via API.
Input Validation: Highlights empty inputs with a subtle shake animation.
Clean, Minimalist Design: Centered layout with soft shadows and rounded corners.
Interactive Transitions: Smooth element reveal when QR code is displayed.

🚀 Technologies Used

HTML5 – Structure and layout
CSS3 – Styling, transitions, and animations
JavaScript (ES6) – Logic and validation.
